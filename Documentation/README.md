# GlucoMan Documentation Index

This folder contains the user documentation for GlucoMan, a diabetes management application.

## User Manuals

### [GlucoMan Operation Manual](GlucoMan_Operation_Manual.md)
Complete operation manual for GlucoMan with detailed descriptions of all features and functions. This comprehensive guide covers:
- Main menu navigation
- Glucose measurement tracking
- Insulin calculation and management
- Meal and food database management
- Recipe management
- Hypoglycemia prediction tools
- Charts and statistics
- Configuration and settings

**Best for:** New users, comprehensive reference, and understanding all features in detail.

### [GlucoMan Quick Reference Guide](GlucoMan_Quick_Reference.md)
Quick reference guide for common tasks and workflows. Includes:
- Quick start instructions
- Common button and icon meanings
- Standard workflows
- Keyboard shortcuts
- Safety reminders
- Troubleshooting tips

**Best for:** Experienced users, quick lookups, and daily use reference.

## Getting Started

If you're new to GlucoMan:
1. Start with the **Quick Reference Guide** for immediate tasks
2. Refer to the **Operation Manual** for detailed information on specific features
3. Review the safety reminders in both documents

## Documentation Source

All user documentation is derived from the tooltips present in the GlucoMan application interface, ensuring consistency between the application and documentation.
